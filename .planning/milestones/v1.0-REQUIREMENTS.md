# Requirements: MIPS CPU Emulator Refactor

**Defined:** 2026-05-15
**Core Value:** Accurately emulate MIPS instruction execution with a functional interactive debugger for local educational use.

## v1 Requirements

### Cross-Platform Build

- [x] **BUILD-01**: Project compiles successfully on Linux without platform-specific errors
- [x] **BUILD-02**: Project compiles successfully on macOS without platform-specific errors
- [x] **BUILD-03**: No Windows-only API imports remain in cross-platform code paths

### Bug Fixes

- [x] **BUG-01**: `load_entry` correctly loads `inst.bin` and `data.bin` without buffer accumulation
- [x] **BUG-02**: No unused imports or dead code warnings on clean build

### Code Quality

- [x] **QUAL-01**: Unused global variables removed or documented
- [x] **QUAL-02**: Debug-only `memcpy` replaced with standard `ptr::copy_nonoverlapping`
- [x] **QUAL-03**: Module naming follows Rust conventions (no raw identifier escapes in public API)

## v2 Requirements

### Instruction Set

- **INST-01**: Implement remaining R-type arithmetic/logic instructions (`add`, `addu`, `sub`, `subu`, `slt`, `sltu`, `or`, `xor`, `nor`, `sll`, `srl`, `sra`, `sllv`, `srlv`, `srav`)
- **INST-02**: Implement branch instructions (`beq`, `bne`, `blez`, `bgtz`, `bltz`, `bgez`, `bltzal`, `bgezal`, `jr`, `jalr`)
- **INST-03**: Implement multiply/divide (`mult`, `multu`, `div`, `divu`, `mfhi`, `mthi`, `mflo`, `mtlo`)
- **INST-04**: Implement memory access (`lb`, `lbu`, `lh`, `lhu`, `lw`, `sb`, `sh`, `sw`)
- **INST-05**: Implement `j`, `jal`, `addi`, `addiu`, `slti`, `sltiu`, `andi`, `xori`

## Out of Scope

| Feature | Reason |
|---------|--------|
| FPU / COP1 | Educational scope limited to integer MIPS core |
| MMU / TLB | Simplified memory model is sufficient |
| Performance benchmarking | Not a production emulator |
| GDB remote protocol | CLI debugger is sufficient for local use |
